# Power BI Portfolio

Welcome to my Power BI Portfolio. This repository is a dedicated space where I showcase my experience in building end-to-end business intelligence solutions, from data modeling and ETL processes in Power Query to data visualization and DAX development.

## Featured Projects

1. **Disney Movies Analysis:** Deep dive into content catalog trends and production metrics using Power BI.
2. **The Game Awards Analysis:** A project focused on nomination patterns, winner distribution, and industry benchmarking.
3. **HBO Content Library:** An analysis of streaming service metrics, focusing on performance, genres, and audience ratings.
4. **WTA Tour 2026 Schedule:** A chronological analysis utilizing custom Gantt chart visualizations for event planning and tracking.

## My Approach to Analytics
My goal is to create reports that are not only aesthetically pleasing but, more importantly, solve real-world problems. Each project in this repository includes:
- The original `.pbix` file.
- Documentation of the data model and logic.
- Insights derived from the data analysis.
