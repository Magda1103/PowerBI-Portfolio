# WTA Tour 2026 Analysis

## Project Overview
This project provides a comprehensive overview of the WTA Tour 2026 schedule. It uses a Gantt chart visualization to track the timeline of tennis tournaments, categorizing them by event level and location to provide a clear seasonal roadmap.

## Key Features
- **Tournament Timeline:** Visualizes 58 tennis events throughout the 2026 season.
- **Category Segmentation:** Filters events by Grand Slam, Team events, WTA 1000, WTA 500, WTA 250, and Year-end championships.
- **Progress Tracking:** Monitors the completion status of the tour (25 out of 58 tournaments completed).
- **Interactive Slicers:** Allows filtering by country, surface type, and specific date ranges.

## Technologies Used
- **Power BI:** Interactive dashboarding and Gantt chart integration.
- **Power Query:** Data shaping for timeline-based events.
- **Custom Visuals:** Utilized a Gantt chart custom visual to manage chronological data efficiently.

## Key Insights
- **Season Flow:** The Gantt chart effectively maps out the transition between major surface types throughout the year.
- **Tour Progress:** As of the current data, the tour is approximately 43% complete, providing a clear snapshot of the remaining schedule.
- **Event Distribution:** The filter panel allows for quick analysis of premium events (e.g., Grand Slams vs. WTA 1000s) across the entire 2026 calendar.

## How to use
You can download the `WTA.pbix` file from this repository and open it in Power BI Desktop to view the full tour schedule and interact with the filters.
